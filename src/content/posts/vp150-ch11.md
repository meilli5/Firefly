## 常用公式
### 动量定义与动能关系（Slide 3）
$$\mathbf{p} = m \mathbf{v}$$
$$K = \frac{1}{2} m v^2 = \frac{\mathbf{p}^2}{2m}$$
- **单位：** $\text{kg}\cdot\text{m/s}$ 或 $\text{N}\cdot\text{s}$
### 牛顿第二定律动量形式（Slide 3 & 5）
$$\mathbf{F}_{\text{net}} = \frac{d\mathbf{p}}{dt} = \frac{d(m\mathbf{v})}{dt}$$
### 冲量与动量定理（Slide 5 & 7）
- **微元冲量：** $d\mathbf{p} = \mathbf{F}_{\text{net}} dt$
- **总冲量线积分（$F-t$ 图像下的面积）：**
$$\mathbf{J} = \int_{t_1}^{t_2} \mathbf{F}_{\text{net}} dt = \Delta \mathbf{p} = \mathbf{p}_2 - \mathbf{p}_1$$
- **等效平均力（Average Force）表示：**
$$\mathbf{J} = \mathbf{F}_{\text{av}} (t_2 - t_1) = \mathbf{F}_{\text{av}} \Delta t \implies \mathbf{F}_{\text{av}} = \frac{\Delta \mathbf{p}}{\Delta t}$$
### 网球反弹平均力求解（Slide 9，一维，初速 $-3\beta$，末速 $2\beta$）
$$\Delta p_x = m(v_{x,2} - v_{x,1}) = m[2\beta - (-3\beta)] = 5m\beta$$
$$F_{\text{avg}} = \frac{\Delta p_x}{\Delta t} = \frac{5m\beta}{\Delta t}$$
### 斜向碰撞平均力分解求解（Slide 10，二维分解）
- **$x$ 轴方向：**
$$F_{\text{avg},x} = \frac{m[v_0 \cos\alpha - (-v_0)]}{\Delta t} = \frac{mv_0(\cos\alpha + 1)}{\Delta t}$$
- **$y$ 轴方向：**
$$F_{\text{avg},y} = \frac{m(v_0 \sin\alpha - 0)}{\Delta t} = \frac{mv_0 \sin\alpha}{\Delta t}$$
- **合平均力大小：**
$$F_{\text{avg}} = \sqrt{F_{\text{avg},x}^2 + F_{\text{avg},y}^2} = \frac{mv_0}{\Delta t} \sqrt{2(1 + \cos\alpha)}$$
### 制动时间与制动距离的数学关系（Slide 12 & 13）
在恒定阻力 $F_{\text{avg}}$ 作用下：
- **制动时间（由动量决定）：** $\Delta t = \frac{\Delta p}{F_{\text{avg}}} = \frac{mv}{F_{\text{avg}}} \propto m \cdot v$
- **制动距离（由动能决定）：** $D = \frac{\Delta K}{F_{\text{avg}}} = \frac{mv^2}{2 F_{\text{avg}}} \propto m \cdot v^2$
### 动量守恒定律（Slide 14 & 16）
对于 $N$ 个质点组成的孤立系统：
$$\frac{d\mathbf{p}_{\text{Total}}}{dt} = \mathbf{0} \implies \mathbf{p}_{\text{Total}} = \sum_{i=1}^N \mathbf{p}_i = \text{常数 (Constant)}$$
### 二维碰撞动量守恒分量方程（Slide 19 & 20）
设 $m_2$ 初始静止（$v_2 = 0$）：
- **$x$ 轴方向：** $m_1 v_1 = m_1 v'_1 \cos\alpha + m_2 v'_2 \cos\beta$
- **$y$ 轴方向：** $0 = m_1 v'_1 \sin\alpha + m_2 v'_2 \sin\beta \implies v'_2 = -\frac{m_1 v'_1 \sin\alpha}{m_2 \sin\beta}$
- **散射角关系：** $\tan\beta = -\frac{v'_1 \sin\alpha}{v_1 - v'_1 \cos\alpha}$
### $m_1 = m_2$ 完全弹性碰撞垂直角推导（Slide 22）
- **动量矢量方程（消去相同质量 $m$）：**
$$\mathbf{v}_1 = \mathbf{v}'_1 + \mathbf{v}'_2$$
- **对矢量方程自乘点积：**
$$v_1^2 = \mathbf{v}_1 \cdot \mathbf{v}_1 = (\mathbf{v}'_1 + \mathbf{v}'_2) \cdot (\mathbf{v}'_1 + \mathbf{v}'_2) = {v'_1}^2 + {v'_2}^2 + 2 (\mathbf{v}'_1 \cdot \mathbf{v}'_2)$$
- **动能守恒方程（消去 $\frac{1}{2}m$）：**
$$v_1^2 = {v'_1}^2 + {v'_2}^2$$
- **对比两式得：**
$$2 (\mathbf{v}'_1 \cdot \mathbf{v}'_2) = 0 \implies \mathbf{v}'_1 \cdot \mathbf{v}'_2 = 0 \iff \mathbf{v}'_1 \perp \mathbf{v}'_2 \quad (\theta = 90^\circ)$$
### 能量变化判据（Slide 27）
定义能量关系式 $K_{\text{before}} + Q = K'_{\text{after}} \implies Q = K' - K$：
- **弹性碰撞：** $Q = 0 \iff K' = K$
- **超级弹性碰撞：** $Q > 0 \iff K' > K$
- **非弹性碰撞：** $Q < 0 \iff K' < K$
- **完全非弹性碰撞：** $Q < 0$ 且碰撞后**两物体速度相同（$\mathbf{v}'_A = \mathbf{v}'_B = \mathbf{v}_{AB}$）**
### 完全非弹性碰撞的共同速度公式（Slide 32）
由动量守恒 $(m_A + m_B)\mathbf{v}_{AB} = m_A \mathbf{v}_A + m_B \mathbf{v}_B$：
$$\mathbf{v}_{AB} = \frac{m_A \mathbf{v}_A + m_B \mathbf{v}_B}{m_A + m_B}$$
### 完全非弹性碰撞（$m_B$ 初始静止 $v_B = 0$）末动能推导（Slide 34 - 36）
- **共同速度：**
$$v_{AB} = \frac{m_A}{m_A + m_B} v_A$$
- **碰撞后末动能：**
$$K_{\text{after}} = \frac{1}{2} (m_A + m_B) v_{AB}^2 = \frac{1}{2} (m_A + m_B) \left( \frac{m_A v_A}{m_A + m_B} \right)^2 = \frac{m_A}{m_A + m_B} \left( \frac{1}{2} m_A v_A^2 \right)$$
- **末动能与初动能比例关系：**
$$K_{\text{after}} = \left( \frac{m_A}{m_A + m_B} \right) K_{\text{before}}$$
### 一维弹性碰撞通用联立方程（Slide 43 - 45）
- **动量守恒：** $m_A v_A + m_B v_B = m_A v'_A + m_B v'_B \implies m_A (v_A - v'_A) = m_B (v'_B - v_B)$
- **动能守恒：** $m_A v_A^2 + m_B v_B^2 = m_A {v'_A}^2 + m_B {v'_B}^2 \implies m_A (v_A^2 - {v'_A}^2) = m_B ({v'_B}^2 - v_B^2)$
- **两式相除得速度和等式：**
$$v_A + v'_A = v_B + v'_B$$
### $m_B$ 初始静止（$v_B = 0$）时的解（Slide 46）
代入 $v_B = 0$ 解得碰撞后各自速度：
- **入射物体 $A$ 的末速度：**
$$v'_A = \left( \frac{m_A - m_B}{m_A + m_B} \right) v_A$$
- **被撞物体 $B$ 的末速度：**    
$$v'_B = \left( \frac{2m_A}{m_A + m_B} \right) v_A$$
### 墙壁获得动能与小球动能比例的极限推导（Slide 49）
墙壁质量 $m_w \gg m$，墙壁获得的动量 $p_w = \Delta p = 2mv$：
$$K_w = \frac{1}{2} m_w v_w^2 = \frac{p_w^2}{2 m_w} = \frac{(2mv)^2}{2 m_w} = \frac{4 m^2 v^2}{2 m_w} = \frac{2 m}{m_w} \left(\frac{1}{2} m v^2\right) = \left(\frac{2m}{m_w}\right) K_b$$
- **动能比例关系：**
$$\frac{K_w}{K_b} = \frac{2m}{m_w} \ll 1 \quad (\text{当 } m_w \gg m \text{ 时趋近于 } 0)$$
### 离散与连续系统的质心公式（Slide 52）
- **离散质点系（$N$ 个质点）：**
$$\mathbf{R}_{\text{cm}} = \frac{\sum m_i \mathbf{r}_i}{M} = \frac{\sum m_i \mathbf{r}_i}{\sum m_i}$$
    - **分量表示：** $X_{\text{cm}} = \frac{\sum m_i x_i}{M}, \quad Y_{\text{cm}} = \frac{\sum m_i y_i}{M}, \quad Z_{\text{cm}} = \frac{\sum m_i z_i}{M}$
- **连续质量分布系统（积分形式）：**
$$\mathbf{R}_{\text{cm}} = \frac{1}{M} \int \mathbf{r} \, dm$$
### 质心运动定理（Slide 51 & 53）
$$\mathbf{F}_{\text{net}} = \sum \mathbf{F}_i = M \frac{d^2 \mathbf{R}_{\text{cm}}}{dt^2} = M \mathbf{a}_{\text{cm}}$$
当 $\mathbf{F}_{\text{net}} = \mathbf{0}$ 时：
$$\mathbf{a}_{\text{cm}} = \mathbf{0} \implies \mathbf{v}_{\text{cm}} = \text{常数} \quad (\text{若初始静止，则 } \mathbf{R}_{\text{cm}} = \text{常数})$$
### 质心运动定理与总动量（Slide 57）
$$\vec{F}_{\text{net}} = \frac{d^2(M\vec{R})}{dt^2} = \frac{d}{dt}\left(\frac{dM\vec{R}}{dt}\right) = \frac{d M\vec{v}_{\text{CM}}}{dt} \implies \vec{P}_{\text{CM}} = \sum_{i=1}^N \vec{p}_i = M \vec{v}_{\text{CM}}$$
当 $\vec{F}_{\text{net}} = \vec{0}$ 时：
$$\frac{d\vec{P}_{\text{CM}}}{dt} = \vec{0} \implies \vec{v}_{\text{CM}} = \text{常数 (Constant)}$$
### 船上人口袋/人在船上运动（Slide 59）
无外力时 $\vec{P}_{\text{CM}} = \vec{P}_{\text{total}} = \vec{0} \implies m \vec{v}_0 + M \vec{v}_b = \vec{0}$：
$$v_b = -\frac{m v_0}{M}$$
>_(负号表示船的退行方向与人的运动方向相反)_
### 炮弹空中爆炸碎片落地点例题（Slide 64）
- **已知：** 炮弹原定落点离发射点距离为 $R$。在顶点/空中爆炸裂为 $m_1$ 和 $m_2$ 两块。碎片 $m_1$ 落于距离发射点 $x_1 = \frac{2R}{3}$ 处。
- **由质心位置不变 $x_{\text{CM}} = R$ 列式：**
$$x_{\text{CM}}(m_1 + m_2) = m_1 x_1 + m_2 x_2 \implies R(m_1 + m_2) = m_1 \left(\frac{2R}{3}\right) + m_2 x_2$$
- **解得碎片 $m_2$ 的落点坐标 $x_2$：**
$$x_2 = R \left( 1 + \frac{m_1}{3m_2} \right) > R$$
- **以质心为坐标原点时的分量形式：**
$$0 = -m_1 x_1 + m_2 x_2 \implies x_1 = -\frac{R}{3}, \quad x_2 = +\frac{R m_1}{3 m_2}$$
### 均匀圆柱体的质心（Slide 67 & 68）
设圆柱体底面积为 $\pi a^2$，长度为 $L$，沿着轴线 $x$ 截取微元 $dx$：
- 微分质量：$dm = \rho \, dV = \rho \pi a^2 dx$
- $x$ 轴质心：
$$x_{\text{CM}} = \frac{1}{M} \int_0^L x \, dm = \frac{1}{M} \int_0^L x (\rho \pi a^2 dx) = \frac{\rho \pi a^2 L^2}{2 M} = \frac{L}{2} \quad \left(\text{因 } M = \rho \pi a^2 L\right)$$
- 由对称性：$y_{\text{CM}} = 0, \quad z_{\text{CM}} = 0$
### 均匀半圆薄片（Half-Disk）的质心（Slide 70 & 71）
- **微元面积与质量：** $dA = 2x \, dy = 2\sqrt{a^2 - y^2} \, dy \implies dm = \rho t (2x \, dy)$
- **总质量与体积：** $V = \frac{1}{2} \pi a^2 t \implies M = \rho V = \frac{1}{2} \rho \pi a^2 t$
- **$y$ 轴质心积分：**
$$y_{\text{CM}} = \frac{1}{M} \int_0^a y \, dm = \frac{1}{M} \int_0^a y \cdot \rho t (2\sqrt{a^2 - y^2}) \, dy = \frac{2\rho t}{M} \int_0^a y (a^2 - y^2)^{1/2} dy$$
    代换积分求得：
$$y_{\text{CM}} = \frac{2\rho t}{M} \left[ -\frac{1}{3}(a^2 - y^2)^{3/2} \right]_0^a = \frac{2\rho t a^3}{3 M} = \frac{2\rho t a^3}{3 \left(\frac{1}{2}\rho \pi a^2 t\right)} = \frac{4a}{3\pi}$$
### 变质量系统（火箭）动量守恒建立（Slide 72）
- **$t$ 时刻总动量：** $p(t) = M v$
- **$t + dt$ 时刻总动量（火箭质量变为 $M + dm$，气体质量为 $-dm > 0$）：**
$$p(t+dt) = (M + dm)(v + dv) + (-dm) v_{\text{fuel}}$$
    代入相对速度 $v_{\text{fuel}} = v + dv - v_{ex}$：
$$p(t+dt) = Mv + M dv + v dm + dm dv - dm(v + dv - v_{ex}) = Mv + M dv + v_{ex} dm$$
- **由 $p(t) = p(t+dt)$ 得火箭运动微分方程：**
$$M dv = -v_{ex} dm \quad (\text{因火箭质量减少，} dm < 0)$$
### 水平推进（Horizontal Propulsion）动量守恒与速度方程（Slide 73 - 76）
- **微分动量守恒方程：**
$$m \, dv = -v_{ex} \, dm \implies \frac{dv}{dt} = -\frac{v_{ex}}{m(t)} \frac{dm}{dt}$$
- **瞬时加速度（因 $\frac{dm}{dt} < 0$，加速度 $a > 0$）：**
$$a = -\frac{v_{ex}}{m(t)} \frac{dm}{dt}$$
- **积分求速度随时间变化关系 $v(t)$：**
$$\int_{v_0}^{v(t)} dv = -v_{ex} \int_{m_0}^{m(t)} \frac{1}{m} \, dm \implies v(t) = v_0 + v_{ex} \ln\left(\frac{m_0}{m(t)}\right)$$
- **燃料耗尽时的最终速度（从静止 $v_0 = 0$ 出发）：**
$$v_{\text{final}} = v_{ex} \ln\left(\frac{m_0}{m_{\text{final}}}\right)$$
### 竖直推进（Vertical Propulsion）动量定理与速度方程（Slide 77 - 80）
- **动量定理（包含重力外力冲量 $-mg \, dt$）：**
$$(mv + m \, dv + v_{ex} \, dm) - mv = -mg \, dt \implies m \, dv = -v_{ex} \, dm - mg \, dt$$
- **竖直瞬时加速度：**
$$a = \frac{dv}{dt} = -\frac{v_{ex}}{m(t)} \frac{dm}{dt} - g$$
- **积分求竖直速度随时间变化关系 $v(t)$：**
$$\int_{v_0}^{v(t)} dv = -v_{ex} \int_{m_0}^{m(t)} \frac{1}{m} \, dm - g \int_0^t dt$$
$$v(t) = v_0 + v_{ex} \ln\left(\frac{m_0}{m(t)}\right) - gt$$
## 易错混淆
- **90° 垂直散射定理成立的前提条件（Slide 21 - 24 适用边界）：**
	- **易错点：** 以为只要是弹性碰撞，碰撞后两物体飞出的方向就一定互相垂直。
	- **纠正：** 垂直散射（$\mathbf{v}'_1 \perp \mathbf{v}'_2$）成立必须同时满足**三个严格前提**：
	    1. **碰撞必须是完全弹性的（Elastic, 动能守恒）**；
	    2. **两物体质量必须严格相等（$m_1 = m_2$）**；
	    3. **被撞目标物体初始必须处于静止状态（$v_2 = 0$）**。
        三者缺一不可，例如当 $m_1 \neq m_2$ 或碰撞是非弹性时，夹角均不等于 90°。
- **单摆碰撞时的“内力与外力”界定（Slide 26 思考题）：**
    - **易错点：** 以为悬挂单摆的绳子拉力（Tension $T$）和重力（Weight $w$）会导致碰撞过程中动量不守恒。
    - **纠正：** 在碰撞的极短时间内，绳子拉力与重力在竖直方向抵消，系统合外力为零；碰撞完全由两球间的相互作用（内力）驱动。真正的外力干扰主要是**空气阻力（Air resistance）**。
- **超级弹性碰撞（Superelastic Collision）的能量来源（Slide 28）：**
    - **易错点：** 以为碰撞后动能增加违反了能量守恒定律。
    - **纠正：** 能量不会凭空产生！动能增加的能量来自于**系统内部预先储存的潜在能量**转化，例如压缩弹簧释放（弹性势能转化）或爆炸（化学能转化）。
- **船上人群走动时“船的位移”与“质心位置”（Slide 53 & 55）：**
	- **易错点：** 误以为人在船上向东走，系统的质心也会向东移动。
	- **纠正：** 人在船上走动靠的是人与船板之间的摩擦力（系统内力）。在没有水阻力（无外力）的前提下，**整体质心在空间中的位置绝对不发生改变**。人向东走会导致船体向西退，使得 $m_{\text{人}} \Delta x_{\text{人}} + m_{\text{船}} \Delta x_{\text{船}} = 0$。
- **变质量系统中的质量变化量 $dm$ 的符号（Slide 72 经典易错点）：**
	- **常见误区：** 认为喷出的气体质量是 $dm$，所以代入时写出正号。
	- **纠正：** 在微积分表达中，$M$ 代表**火箭自身的质量**。由于火箭不断喷气，火箭质量随时间减少，因此其微分量 **$dm < 0$ 必为负数**！喷出的气体质量则应记为正数 **$-dm > 0$**。
